[.git]이 자동으로 생성되는데 이 폴더는 git으로 생성한 버전들의 정보와 원격 저장소 주소가 들어있어 이 폴더를 로컬 저장소라고 부른다.

밑에는 대충 수정

Git: 버전 관리 시스템
GitHub : git으로 관리하는 프로젝트를 올려둘수 있는 사이트
GUI : 그래픽 유저 인터페이스, 마우스로 클릭해서 사용
CLI: 커맨드 라인 인터페이스, 명령어를 하나씩 입력하는 방식
Git Bash : CLI 방식으로 Git을 사용할수 있는 환경
commit : 버전관리를 통해 생성된 파일, 혹은 그 행위를 의미
log  명령어: 지금까지 만든 커밋을 모두 확인
checkout : 원하는 지점으로 파일을 되돌릴수 있다.
로컬저장소: Git으로 버전관리하는 내컴퓨터안의 폴더
원격저장소 : GitHub에서 협업하는 공간(폴더)를 의미
Repository : 원격저장소
Push : 로컬저장소의 commit을 원격저장소에 올리는것
pull : 원격저장소의 커밋을 조컬저장소에 내려받는것.
md : 확장자는 텍스트 문서를 적기위한 파일, 마크다운문서. 프로그래밍 언어와 같이 간단한 문법이 적용됨.
